基于Hadoop、Hive和Spark的电商用户行为可视化分析系统

E-commerce user behavior visualization analysis system based on Hadoop, Hive and Spark

数据分析维度

分别从平台维度、用户维度、商品维度分析平台维度通过对电商流量维度的分析，可以了解到用户流量的相关信息，比如用户活跃时间、用户流量漏斗转化率，了解用户的流量趋势。用户维度通过对用户行为的分析，可以了解用户四种行为数据，用户留存率等。通过对用户价值进行分析，使用RFM模型，对用户进行分类，找出有价值的用户，对用户进行精细化运营。商品维度通过对商品品类、商品成交量等指标的分析，探索用户对商品及商品品类的偏好，了解商品销售的规律。

VMware
VMware 是一款虚拟化软件。它可以在一台物理计算机上创建多个虚拟计算机环境，每个虚拟环境可以安装不同的操作系统，方便进行软件测试、开发多系统环境应用等，同时也有助于提高硬件资源的利用率。
Xshell
Xshell 是一款功能强大的终端模拟软件。主要用于连接远程服务器，通过它你可以在本地计算机上以命令行的方式对远程服务器进行管理和操作，如执行命令、上传下载文件等。
MySQL
MySQL 是一种关系型数据库管理系统。它用于存储和管理大量结构化数据，支持 SQL 查询语言，可以高效地进行数据的插入、查询、更新和删除等操作，广泛应用于各种 Web 应用和企业级软件开发中。
IntelliJ IDEA
IntelliJ IDEA 是一款集成开发环境，主要用于 Java 开发，但也支持其他编程语言。它提供了丰富的代码编辑、调试、测试等功能，能够提高开发效率，帮助开发者更轻松地编写高质量的代码。
Hive
Hive 是基于 Hadoop 的数据仓库工具。它可以将结构化的数据文件映射为数据库表，并提供类 SQL 的查询语言 HiveQL，使得不熟悉 MapReduce 的用户也能方便地对大规模数据进行查询和分析。
Hadoop
Hadoop 是一个开源的分布式计算平台。主要由 HDFS（分布式文件系统）和 MapReduce（分布式计算框架）组成。它可以处理大规模数据，将数据存储在多个节点上，并通过并行计算提高数据处理的速度和效率。
Spark
Spark 是一种快速、通用的大数据处理引擎。相比 Hadoop 的 MapReduce，Spark 具有更高的计算性能和更丰富的数据分析功能。它支持内存计算，可以大大减少数据处理的时间，适用于实时数据分析、机器学习等场景。
Node.js
Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。它允许开发者使用 JavaScript 编写服务器端代码，实现高效的网络应用开发。Node.js 具有非阻塞、事件驱动的特点，能够处理大量并发连接，适用于构建实时性要求高的 Web 应用和微服务架构。
